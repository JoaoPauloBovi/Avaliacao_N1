Avaliação N1 - Algoritmos e Programação

Este repositório centraliza o desenvolvimento técnico e exercícios práticos realizados para a Avaliação N1 do curso de Análise e Desenvolvimento de Sistemas. O objetivo é consolidar os fundamentos de lógica e a base das linguagens estruturadas e orientadas a objetos.
Objetivos Técnicos

    Implementação de lógica de programação e algoritmos.

    Estudo de sintaxe e estruturas fundamentais em C e C++.

    Gerenciamento de memória e manipulação de variáveis.

    Versionamento de código com Git aplicado a contextos acadêmicos.

Estrutura do Repositório

    Exercícios de Fixação: Códigos desenvolvidos para validar conceitos teóricos das aulas.

    Projetos de Avaliação: Implementações práticas voltadas para a nota da N1.

Tecnologias e Ferramentas

    Linguagens: C e C++.

    Ambiente de Desenvolvimento: IDE VSCode e Git para versionamento.

    Professor: Oswaldo Peres.
